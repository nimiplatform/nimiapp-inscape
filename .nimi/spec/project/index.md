# Inscape domain

Inscape (心相) is a fully local, open-source, 18+ Jungian-typology desktop app.
It turns the cognitive-function stack (with Beebe archetypes) into daily-life
application, and treats type as a moving probability distribution — never a
fixed label.

Product positioning, target user, value propositions, anti-targets, and the
twelve locked decisions are authored in the nimi-realm topic
`2026-05-27-inscape-app-design-and-audit-closure` (`product-manual.md` /
`user-storyboard.md`). This kernel carries the normative implementation
contracts that downstream code must respect.

See `kernel/index.md` for the authority map.
